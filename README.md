# Dotfiles

## Requirements

Make sure you have installed these following packages

### Git

```
pacman -S git
```

### Stow

```
pacman -S stow
```

## Installation

Clone this repo using git

```
$ git clone https://github.com/abdurrahmanr/dotfiles.git
$ cd dotfiles
```

then use stow to create symlinks

```
$ stow .
```
